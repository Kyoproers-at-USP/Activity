## Git操作
### リポジトリのクローン
リモート（インターネット上）に存在するリポジトリをローカルの環境にコピーすることを"クローン"という．
```bash
# リポジトリをクローンするディレクトリへ移動
$ cd ~/path/to/your-directory

# リポジトリのクローン
$ git clone <remote-repository-url>

## ここからはオマケ
# 必要であれば変数として登録
$ echo "GITHUB_DIR='$(pwd)'" >> ~/.bashrc

# 設定ファイルの再読み込み
$ source ~/.bashrc
```
### 基本操作
```bash
# 特定のファイルをステージング
$ git add <file_name>
# ステージングしたファイルをコミット
$ git commit -m 'Initial commit'
# リモートリポジトリの<branch-name>ブランチにPush
$ git push origin <branch-name>
# リモートリポジトリの<branch-name>ブランチの内容をローカルリポジトリに
$ git pull origin <branch-name>
```
### ブランチ関係
```bash
# 新しいブランチ作成
$ git branch <branch-name>
# ブランチ変更
# ローカルに存在しなくても，リモートに存在している場合は，
# fetch後に同様のコマンドを実行することで作成可能
$ git switch <branch-name>
# 新しいブランチ作成&変更
$ git switch -c <branch-name>
```
### その他
```bash
# リモートリポジトリの内容をダウンロード
$ git fetch origin
# 指定したブランチの内容を現在のブランチにマージ
$ git merge <branch-name>
```
