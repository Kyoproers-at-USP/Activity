# 本ディレクトリについて
本ディレクトリはサークル活動に関するファイルをまとめたものになります．
活動期間・活動内容を視覚的に分かりやすくするため、ファイル共有時は下記の規則に従ってください．
## ファイル共有時のルール
1. ファイル数が1つの時
    * `<活動開始年月>_<活動名>`という形の snake_case 記法で命名してください
    * 例）`202504_パズルゲーム制作.md`
2. ファイル数が2つ以上の時
    * `<活動開始年月>-<活動名>`という形の kebab-case 記法で命名した専用ディレクトリを一つ作成し、内部に複数ファイルを保管してください
    * ディレクトリ内に保持するファイル名は自由とします
